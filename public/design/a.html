What this direction is

Dark base. Electric accents. Generous glow. Motion as a primary design surface. Meant to feel like the apps a 16-year-old actually opens at night.

Palette
Near-black base, cyan + purple + pink neon for moments, soft gradient blooms.
Type
Inter at heavy weights (800–900) for impact, JetBrains Mono for labels and microcopy. Tight tracking, gradient highlights on punch words.
Motion
Gradient drift, orb pulse, glow auras, slow background blooms. Always optional via reduced-motion.
Character (Kai)
An orb with a gradient core and a soft aura. Animated breathing. No face, no anthropomorphism — Kai is a presence, not a mascot.
Copy voice
Short, punchy, slightly cocky. "An AI that actually gets it." "Best streak you've ever had." Confident but never preachy.
Reads as
Discord meets Linear meets a teen producer's project file. Reads premium without reading corporate.
